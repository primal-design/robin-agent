# Primal Portfolio

Static multi-page website ready for Netlify.

## Main pages

- `index.html` - Home
- `about.html`
- `services.html`
- `fen.html`
- `case-studies.html`
- `employers.html`
- `hire-me.html`
- `contact.html`

## Shared assets

- `style.css`
- `script.js`
- `netlify.toml`

## Deploy to Netlify

1. Open [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag the whole `primal-portfolio` folder into the page
3. Netlify will publish the site instantly

## Replace before publishing

- `LinkedIn URL` placeholder in `contact.html`
- `Download CV` placeholder links in `hire-me.html` and `contact.html`
- Legal placeholder copy in `privacy.html`, `terms.html`, `cookie-settings.html`, and `data-rights.html`
