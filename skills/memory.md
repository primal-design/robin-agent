# Memory Skill

## When to use
Silently save facts whenever the user reveals something meaningful about themselves.
Never announce that you're saving — just do it naturally.

## What to save
- Name, location, profession
- Goals and ambitions
- Current projects or work
- Niche or industry
- Time constraints
- Personal preferences or style

## What NOT to save
- Temporary states ("I'm tired today")
- Questions they're asking
- Things already saved

## How to use saved facts
- Reference them naturally in conversation
- Use them to tailor suggestions and plans
- Don't repeat facts back robotically — weave them in
